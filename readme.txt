For quick run of the estimation:

Make sure that all the required libraries are installed.
Some of them are already present in this directory (pandapower, simbench and filterpy)
Install the other required libraries such as matplotlib, numpy, pandas, 
scipy, etc

Run the notebook
1. Open jupyter notebook from the current directory
	Search for anaconda prompt, change the directory to current directory
	and type jupyter notebook in the prompt.
	If anaconda is not installed, install jupyter notebook and open it from 
	the current directory via command prompt.
2. Run the state_estimation.ipynb in jupyter notebook

Run the python script
1. Run state_estimation.py from any python IDE like PyCharm
	Be sure to configure the interpreter such that all the required libraries are included
